This is a Vite + React + Tailwind starter for the Weather Dashboard capstone.

## Quick start
1. copy `.env.example` -> `.env` and set `VITE_OPENWEATHER_API_KEY`
2. npm install
3. npm run dev

## Deploy to Vercel
- Push to GitHub, import into Vercel and set env var `VITE_OPENWEATHER_API_KEY`.
